Font Family Examples

Serif Font Example

This paragraph is styled with a serif font family. Serif fonts have small lines or strokes at the ends of each character. They are often used in traditional and formal contexts, such as in books and newspapers. This text is using "Times New Roman", Times, serif.

Sans-serif Font Example

This paragraph is styled with a sans-serif font family. Sans-serif fonts do not have the small lines or strokes at the ends of characters. They are clean, modern, and widely used for web content. This text is using "Helvetica Neue", Arial, sans-serif.

Monospace Font Example

This paragraph is styled with a monospace font family. Monospace fonts have characters that occupy the same amount of horizontal space, making them ideal for code and technical documents. This text is using "Courier New", Courier, monospace.

Cursive Font Example

This paragraph is styled with a cursive font family. Cursive fonts are designed to mimic handwriting and often include decorative elements. They are used for artistic and personal touches. This text is using "Brush Script MT", cursive.

Fantasy Font Example

This paragraph is styled with a fantasy font family. Fantasy fonts are decorative and unconventional, often used for special effects or themed designs. They stand out and are not typically used for body text. This text is using Impact, fantasy.
